Using setTimeOut

The setTimeout() method calls a function after a number of milliseconds. Like 1 second = 1000 milliseconds.

Notes:
The setTimeout() is executed only once.
If you need repeated executions, use setInterval() instead. Use the clearTimeout() method to prevent the function from starting.
To clear a timeout, use the id returned from setTimeout():

clearTimeout():
The global clearTimeout() method cancels a timeout previously established by calling setTimeout().
